# Maharashtra-District-Demographic-Urban-Insights-Dashboard
This Power BI dashboard provides insights into demographic and urban statistics of Maharashtra districts.  It helps analyze urban population, literacy rate, and administrative division distribution.

🛠️ Tools & Technologies

Power BI

Data Modeling

DAX (Data Analysis Expressions)

📊 Dashboard Features

Interactive slicers for District Name and Administrative Division.

Pie chart showing distribution of districts by division.

Bar chart representing average urban population percentage by district.

Visualization of literacy rate comparison across districts.

Analysis of urban population based on number of talukas.

📈 Key Insights

Mumbai City has the highest literacy rate.

Thane shows a significantly high urban population.

Urbanization varies across districts.

Administrative divisions have a balanced distribution of districts.

🎯 Objective

To analyze demographic patterns and urbanization trends in Maharashtra.

To provide meaningful insights using data visualization.

📌 Conclusion

The dashboard transforms raw data into clear and actionable insights.

It enables better understanding of district-level demographic trends.
